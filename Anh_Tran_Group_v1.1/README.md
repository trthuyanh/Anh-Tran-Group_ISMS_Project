3. The CloudFlow Product

CloudFlow is Tran Anh Group's flagship product, a cloud-based project management platform designed for mid-sized companies (50-500 employees) who need more than basic task tracking but find enterprise solutions too complex or expensive.

Key Features:

Visual project planning with Gantt charts and Kanban boards Real-time team collaboration and file sharing Customizable workflows and automation Time tracking and resource allocation Integrations (Slack, Google Workspace, Microsoft 365) Mobile apps (iOS and Android) Advanced reporting and analytics API for custom integrations

Target Market: Mid-market companies ($10M-$500M revenue) in project-intensive industries. Typical  customer: 100-200 employees, 20-50 active CloudFlow users
